Intersystems Caché database driver

History: Intersystems have created database, using multidimensional cube for storing data. It provides SQL, Object and direct interface to data (to this cube). Ruby lacks all support for it (only through ODBC). I've implemented object interface.

It is now possible to directly use Caché classes. Look at this Caché screencast. After doing all, what is there show, it is possible to use class Person from tests.
